# Student Management System

A modern, premium Student Management System built with Flutter and Firebase.

## ✍️ Author
**Samindi  Hansika**

## 🚀 Features
- **Ultra-Premium UI**: Glassmorphic elements, gradients, and smooth animations using Material 3.
- **Student Directory**: View all registered students in a clean, card-based list.
- **CRUD Operations**: Complete Create, Read, Update, and Delete functionality.
- **Firebase Integration**: Real-time data synchronization with Cloud Firestore.
- **Success Notifications**: Beautiful floating snackbars for user feedback.

## 🛠️ Technology Stack
- **Framework**: [Flutter](https://flutter.dev)
- **Backend**: [Firebase Firestore](https://firebase.google.com/docs/firestore)
- **Typography**: Inter / Roboto
- **Design System**: Material 3 (Customized)

## 📋 How to Run
1. Ensure you have Flutter installed.
2. Clone the repository.
3. Run `flutter pub get` to install dependencies.
4. Execute `flutter run` (Use `-d chrome` for web).
